# Lagunas Claras

This project is a web application for the company Lagunas Claras, dedicated to the maintenance, recovery, and preservation of lakes, lagoons, and water bodies. It uses modern technologies such as Astro, TailwindCSS, and other web ecosystem tools to provide an attractive and functional visual experience.

- Astro
- TailwindCSS
- TypeScript

## Structure

- Informational sections (about us, services, clients, contact)
- Animations and responsive design
